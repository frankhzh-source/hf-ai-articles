# 维度 4 · 面部细节 — 中文 Prompt 模板（LibTV 图像/视频节点专用）

## 核心目标
固定人物面部六要素（脸型 / 眉眼 / 鼻 / 唇 / 妆容 / 皮肤），让每个镜头生成的面部细节完全一致，消除「换镜如换脸」问题。

---

## 一、面部六要素分解

| 要素 | 写什么 | 不写什么 | 效果对比 |
|------|-------|---------|---------|
| **脸型** | 鹅蛋脸、圆脸、瓜子脸、方脸 | 小脸、精致脸 | 脸型不写 → 每镜脸型随机变化 |
| **眉毛** | 自然弯眉、平直眉、柳叶眉、落尾眉 | 好看的眉毛、细眉 | 眉形不稳 → 每镜眉毛弧度不同 |
| **眼睛** | 杏眼、丹凤眼、细长眼、圆眼，睫毛长度 | 大眼睛、有神 | 不写眼型 → 镜头切换眼距变近/变远 |
| **鼻子** | 直鼻梁、小巧鼻尖、鼻翼窄 | 高鼻梁（过于抽象） | 不写 → 鼻梁在侧脸时忽高忽低 |
| **嘴唇** | 上唇薄下唇厚、M形唇、嘴角微扬 | 性感、饱满 | 没写 → 唇色每镜不同，口红色号不稳定 |
| **皮肤** | 暖白偏黄、自然肤色、哑光肌、偏干性 | 好皮肤、白 | 不写 → 曝光不同时色彩断层 |

## 二、电商家居服「面部细节」固定参数

### 2.1 基准妆容（固定块，每镜复制）

```
自然裸妆，豆沙色口红，哑光质地
眉毛自然有弧度，不粗不细，深棕眉色
眼妆清淡，内眼线，睫毛自然微翘
鼻影柔和，高光不油腻
皮肤暖白偏黄，哑光质感，不磨皮，保留皮肤纹理
整体妆感清透自然，不像化妆师精修
```

### 2.2 六要素填写对照

```
脸型：鹅蛋脸
眉毛：自然弯眉，深棕色，粗细适中
眼睛：杏眼，内眼角略尖，睫毛自然微翘
鼻子：直鼻梁，鼻翼窄，鼻头小巧
嘴唇：上唇薄下唇厚，豆沙色口红，哑光质地
皮肤：暖白偏黄，哑光，保留肌肤纹理，不磨皮
```

### 2.3 填错 vs 填对

| 要素 | ❌ 坏写法 | ✅ 好写法 |
|------|---------|---------|
| 脸型 | 小脸、精致 | 鹅蛋脸，下颌线条柔和 |
| 眉毛 | 好看的眉毛 | 自然弯眉，深棕色，粗细适中 |
| 眼睛 | 大眼睛有神 | 杏眼，内眼角略尖，双眼皮自然 |
| 鼻子 | 高鼻梁 | 直鼻梁，鼻翼窄 |
| 嘴唇 | 饱满嘴唇 | 上唇薄下唇厚，豆沙色哑光 |
| 皮肤 | 好皮肤 | 暖白偏黄，哑光，微纹理 |
| 妆容 | 淡妆即可 | 自然裸妆，豆沙唇，内眼线，不磨皮 |
| 睫毛 | 长睫毛 | 睫毛自然微翘，不过分浓密 |
| 双眼皮 | 双眼皮 | 自然扇形双眼皮，宽度适中 |
| 下巴 | 尖下巴 | 下巴微尖，线条柔和不突兀 |

## 三、六要素固定块（可直接复制）

```
鹅蛋脸，自然弯眉深棕色，杏眼内眼角略尖睫毛微翘，直鼻梁鼻翼窄，上唇薄下唇厚豆沙色哑光，暖白偏黄哑光肌保留纹理不磨皮
```

> 这行就是一个完整的面部固定块。**每镜提示词直接复制粘贴进去，不要改。**

---

## 四、每镜妆容一致性：翻车与修法

| 翻车 | 根因 | 修法 |
|------|------|------|
| 第一镜红唇，第二镜变裸唇 | 唇色没写入固定块 | 每镜提示词开头写「豆沙色哑光口红」 |
| 特写镜头皮肤像开了美颜 | 没写「保留皮肤纹理，不磨皮」 | 加「保留纹理，哑光肌」 |
| 半身图面部和脖子色差大 | 没写统一肤色 | 加「暖白偏黄，面部和颈部肤色一致」 |
| 不同光照下图脸型变了 | 脸型没写 | 加「鹅蛋脸」 |
| 眉毛一镜粗一镜细 | 眉形没固定 | 加「自然弯眉，深棕色，粗细适中」 |
| 眼距忽远忽近 | 没写眼型 | 加「杏眼，眼距适中」 |
| 高光太油像出油 | 没控制光泽 | 加「哑光感，不过度高光」 |

## 五、LibTV 使用方案

### 5.1 图像节点

在每张图的提示词中，把面部固定块插在「人物身份」和「服装锁定」之间：

```
[人物身份] + [面部固定块] + [服装锁定] + [视角/表情/光照/构图]
```

**实战（达芬奇密码洛神玫瑰）**：

```
亚洲女性，28岁，电商家居服模特
鹅蛋脸，自然弯眉深棕色，杏眼内眼角略尖睫毛微翘，直鼻梁鼻翼窄，上唇薄下唇厚豆沙色哑光，暖白偏黄哑光肌保留纹理不磨皮
穿着参考图中洛神玫瑰青果领家居服
正面半身照，面向镜头，嘴角微扬
暖光3000K，45度右侧柔光
背景暖白简约卧室，略微虚化
高清，细节清晰，面部清晰，妆容一致
```

### 5.2 特写镜头的额外处理

当镜头推进到面部特写（胸像以上），需要增加：

```
面部细节清晰，眼睫毛根根分明，嘴唇纹理可见
皮肤毛孔保留自然质感，不过度磨皮
眉毛每一根清晰可见
瞳孔反光自然柔和
```

### 5.3 参考图节点

上传一张「维度1主视觉肖像」高清面部特写作为参考图，作用是帮模型锁定：

| 参考图提供的信息 | 提示词里写不到的部分 |
|-----------------|-------------------|
| 面部结构比例 | 颧骨高度、下颌角度、额头宽度 |
| 肌肉走向 | 笑纹、法令纹位置 |
| 皮肤真实质感 | 毛孔大小、红血丝分布 |
| 微结构 | 唇纹、眼纹、眉骨高度 |

**参考图权重建议**：0.8（过高会把光照和构图一起锁死）

## 六、视频节点的面部要求（即将使用）

视频生成面部时，除了固定块还要额外处理：

```
[面部固定块]
面部连贯无变形
每帧妆面一致
眼神自然不漂移
表情过渡平滑不抽搐
```

### 视频翻车 & 修法

| 翻车 | 修法 |
|------|------|
| 人物转头时五官错位 | 加「面部轮廓稳定不扭曲」，用参考图加权 |
| 说话时嘴型僵硬 | 加「自然说话嘴形，唇部动作流畅」 |
| 眨眼频率不自然 | 加「自然眨眼频率，不频繁不瞪眼」 |
| 侧脸时面部扁平 | 加「侧脸时鼻梁和颧骨轮廓分明」 |
| 表情过渡生硬 | 写表情变化过程（见维度3），不要只在开头写表情 |

## 七、检查清单

每张图生成后，逐项检查：

- [ ] 脸型是否和参考图一致？
- [ ] 眉毛弧度、粗细、颜色是否固定？
- [ ] 眼型、眼距是否稳定？
- [ ] 鼻梁高度、鼻翼宽度是否一致？
- [ ] 唇色是否每镜相同？
- [ ] 皮肤质感是否有磨皮感？
- [ ] 肤色是否和参考图一致（无偏绿/偏红）？
- [ ] 面部和颈部有无色差？
- [ ] 特写镜头皮肤纹理是否保留？
- [ ] 多角度（正/侧/背）下脸型是否一致？

**任一不通过 → 六要素固定块整段复制去检查，不要只改一个词。**
