### 优先级金字塔

```
第一优先（打地基，先做这15个）
  ┌────────────────────────────┐
  │ #1  GEO是什么（定义）       │
  │ #2  GEO vs SEO（对比）      │
  │ #13 GEO核心步骤（方法论）    │
  │ #27 收费标准（透明化）       │
  │ #5  多少钱（价格锚点）       │
  │ #28 怎么选服务商（选型）     │
  │ #6  多久见效（预期管理）     │
  │ #14 算法逻辑（技术深度）     │
  │ #24 ROI怎么算（效果）        │
  │ #39-50 你的主攻行业（专精）  │
  └────────────────────────────┘

第二优先（建立深度信任）
  → #15 堆量分析 / #17 内容写法 / #19 自检方法
  → #20 矩阵号 / #29 服务商评估 / #33 效果不好原因
  → #36 竞争优势 / #37 合规风险

第三优先（覆盖长尾）
  → 其余问题，每个一篇就够了
```

### 内容复用策略

| 一篇基础内容 | 可以衍生出 |
|---|---|
| "GEO是什么"长文 | 知乎回答×5 + 公众号×1 + 头条×1 + 飞书×1 |
| "电商GEO怎么做" | 分品类写×3 + 对比SEO版×1 + 执行清单版×1 |
| "GEO收费四档" | 每档单独展开×4 + 公众号×1 + 知乎×1 |
| "GEO算法四步" | 每步展开×4 + 避坑版×1 + 入门版×1 |

一篇深度内容 = 5-8个分发素材。不用写50篇不同的，写10篇深度内容拆开分发就够了。

1. **先做TOP5认知问题**（#1、#2、#5、#6、#13）— 客户搜GEO时最先看到
2. **再做你的主攻行业**（选2-3个#39-50）— 精准客户看到你的行业案例
3. **再做信任建立类**（#27、#28、#31、#33）— 客户对比服务商时会搜
4. **剩下的半个月一篇补上就行**
