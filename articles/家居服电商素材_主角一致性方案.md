# 家居服电商素材 · 主角一致性方案

## 一、人物资产准备

### 1.1 模特定妆照（拍摄标准）

**需提前准备的6张素材：**

| 角度 | 用途 | 拍摄要点 |
|------|------|---------|
| 正面站立 | 全身基准 | 双臂自然下垂，完整展示家居服正面版型 |
| 正面半身 | 面容基准 | 与全身照同光源、同表情，用于LoRA训练 |
| 侧身45° | 侧面版型 | 展示家居服腰线、袖子垂坠效果 |
| 侧身90° | 背部+S曲线 | 展示家居服后背设计、裤子后身版型 |
| 手持物展示 | 道具确认 | 手拿水杯/手机/抱枕等配合道具手势 |
| 坐姿/自然姿态 | 生活感姿态 | 坐在床边/沙发，展示家居服坐姿版型 |

**服装状态固定：**
- 家居服必须提前熨烫平整，拍摄前确认无褶皱、无线头
- 扣子/系带位置固定（第一颗扣子扣/不扣，腰带系法）
- 袖口卷边状态固定（挽起2圈 or 自然垂下）

### 1.2 模特形象固定清单

```
发型：{黑长直/披肩/发尾微卷至胸口/左侧刘海}
妆容：{裸妆/豆沙色唇/无明显眼影/眉毛自然}
耳饰：{无} 或 {金色小圈直径1.5cm}
手部：{无美甲/指甲修剪整齐/无戒指}
鞋履：{米白棉拖鞋/无图案}
```

> 电商模特换造型成本低——定好就不改，全片统一。

## 二、服饰描述规范（关键）

### 2.1 服饰描述的结构化模板

每个镜头必须含以下**固定描述块**，一字不改：

```
{套件: 长袖衬衫式家居服套装}
{上衣: 翻领/V领/衬衫领 + 粉色滚边 + 花卉印花图案 + 左胸无口袋}
{下裤: 松紧腰 + 直筒裤型 + 全长至脚踝 + 同款花卉印花}
{腰带: 同面料/宽3cm/系单蝴蝶结于腰前正中}
{扣子: 珍珠白/直径1cm/5颗/全部系好}
{面料: 缎面丝绸/中等光泽/垂坠感好}
{滚边: 粉色/宽0.5cm/领口+袖口+前襟边缘}
{印花: 大朵粉色山茶花/红橙粉渐变/横向排列/前幅下摆为主}
```

> **铁律：** 全部15个镜头的描述文本必须**绝对一致**（Ctrl+C/V），一个字不能改。改一个字模型就可能理解为"不同衣服"。

### 2.2 各场景下的「面料」描述优化

不同模型对材质关键词的敏感度不同，按模型选择：

| 模型 | 面料关键词推荐 | 不推荐 |
|------|-------------|--------|
| Seedance 2.5 | satin silk fabric, subtle sheen, smooth draping | "高级感"（太抽象）|
| 可灵 1.6 | 丝绸面料, 缎面光泽, 垂坠感好 | "柔软丝滑"（模型不识别）|
| Kling 1.6 | silk pajama set, satin texture, light-reflecting | "亲肤"（模型无法渲染）|

通用描述（中英文双语最稳）：
```
面料: 缎面丝绸  satin silk fabric
光泽: 中等缎面光泽  medium satin sheen
垂坠: 自然顺滑垂坠  smooth natural draping   
厚度: 薄至中等  light to medium weight
```

### 2.3 印花描述的精细度

| 写法 | 效果 | 建议 |
|------|------|------|
| ❌ "花卉图案" | 每次生成不同花 | 太模糊，不采用 |
| ✅ "大朵粉色山茶花/红橙粉渐变" | 锁定花型+颜色 | 采用 |
| ✅ "横向排列/前幅下摆为主" | 锁定位置 | 建议加 |
| ⭐ "参考图ID: print_ref.jpg" | 直接锁死 | 最推荐 |

## 三、配饰与道具描述

### 3.1 配饰清单（每镜固定）

```
【配饰-固定】无耳饰/无项链/无手镯/无戒指/棉拖鞋米白色
```

> 家居服场景配饰越少越好——减配饰=减一致性风险。

### 3.2 道具出现规则

| 道具 | 出现位置 | 约束 |
|------|---------|------|
| 水杯（白瓷） | 开场/结尾 | 右手持/杯柄朝右 |
| 手机 | 仅中场 | 左手持/屏幕朝内 |
| 抱枕 | 仅坐姿 | 浅灰蓝色/放在腿上 |

**道具管理铁律：**
1. 每件道具只在**固定镜头**出现，不跨镜
2. 道具描述统一写在分镜表里：`{道具: 白瓷杯/右手持/杯柄朝右}`
3. 不需要道具的镜头，显式写：`{道具: 无}`（防止模型凭空生成）

## 四、分镜一致性检查表（出片前必过）

### 4.1 第一关：文本一致性

逐镜比对以下字段，全部15镜必须完全一致：

- [ ] 模特形象描述（发型/妆容/耳饰/手部）
- [ ] 服装套件描述（套件名称/上衣/下裤/扣子）
- [ ] 面料描述（材质/光泽/垂坠）
- [ ] 印花描述（花型/颜色/排列/位置）
- [ ] 配饰描述（全部配饰清单）

> 不一致 → 复制前一镜的描述块覆盖，不要手动重打。

### 4.2 第二关：参考图一致性

- [ ] 每个镜头指定了参考图ID
- [ ] 同一角色的全部镜头使用同一组参考图
- [ ] 特写镜头（领口/印花/袖口）各有独立细节参考图
- [ ] 面料特写单独有一张面料质感的参考图（对丝绸光泽渲染至关重要）

### 4.3 第三关：抽检标准

**首帧抽检：**
- 生成每个镜头的第一帧（图片模式），不做视频
- 全部15帧排成网格，逐行比对
- 检查项：服装颜色、印花位置、发型长度、耳饰有无
- 通过率 ≥ 90%，不合格镜头重新调Prompt

**视频成片后抽检：**
- 同场景相邻镜头各取中间帧对比
- 道具位置连续性检查
- 连续播放3帧/s，无"跳变"即通过

## 六、一句话总结

> **服装描述：用结构化模板锁死，15镜完全一致。**
> **面料关键词：按模型选，中英文双语。**
> **配饰：能不加就不加，必须加的锁死在固定镜头。**
> **道具：每件限定1-2个镜头，不跨镜。**
> **最终检查：15个首帧排成网格，一眼看出谁走样。**
