# 维度 2 · 三视图 — 中文 Prompt 模板（LibTV 图像节点专用）

## 核心目标
为角色建立正面、侧面、背面三个角度的统一视觉基准。三视图不是「一张图里拼三个角度」，而是**同服装 / 同光源 / 同人物 的三张独立图**。

---

## 一、截图问题诊断

你这张图的问题很典型：

- **正脸清晰，侧面、背面被压暗 / 模糊**
- 文生图模型（Seedream 4.5）默认只理解一个主视觉，它把侧面和背面当成「背景」或「次要对象」处理了
- 同图里写「正面、侧面、背面」= 让模型做一件它不擅长的事

在 LibTV 图像节点里，正确做法不是让一张图出三视图，而是**拆成 3 个图像节点，每个节点分别生成一个角度**，再用「主视觉肖像」作为参考图锁住人物一致性。

---

## 二、LibTV 三视图提示词范式

```
[主体身份] + [服装锁死] + [视角固定] + [光照统一] + [构图稳定] + [输出要求]
```

每个角度的前四层（主体、服装、视角、光照）要保持完全一致，只有**视角**和**构图**变化。

## 五、LibTV 节点编排建议

### 节点结构

```
图像节点 1：正面全身图
  → 模型：Seedream 4.5
  → 比例：9:16
  → 参考图：上传「维度1主视觉肖像」
  → 提示词：3.1 正面图
  → 负面提示词：第四节

图像节点 2：左侧面 / 45° 图
  → 模型：Seedream 4.5
  → 比例：9:16
  → 参考图：上传「维度1主视觉肖像」
  → 提示词：3.2 左侧面图
  → 负面提示词：第四节

图像节点 3：背面图
  → 模型：Seedream 4.5
  → 比例：9:16
  → 参考图：上传「维度1主视觉肖像」
  → 提示词：3.3 背面图
  → 负面提示词：第四节
```

### 参考图用法

- 把「维度1主视觉肖像」那张图作为每个图像节点的 **参考图（Image Prompt / Reference Image）**
- 参考图权重：0.8 - 0.9，不要拉到 1.0（会压垮原提示词）
- 如果 LibTV 支持 ControlNet，侧视图可用 OpenPose 或 depth 控制姿态，但不要同时绑正面姿态

## 六、视角用词表

| 角度 | 推荐写法 | 不推荐写法 |
|------|---------|-----------|
| 正面 | 正面站立，面向镜头 | 正面，正脸 |
| 45° 侧 | 左侧面45度站立，面部转向正前方 | 侧脸，半侧面 |
| 90° 侧 | 身体侧面朝向镜头，面部保持轮廓 | 纯侧面 |
| 背面 | 背面站立，背部朝向镜头 | 背面，背影 |

> 为什么写「面部转向正前方」？避免侧视图把脸完全转过去，导致看不到面部锚点。

## 七、三视图统一参数表

这六项必须在三个图里保持一致，否则检核不通过：

| 参数 | 固定写法 | 说明 |
|------|---------|------|
| 人物身份 | 亚洲女性，28岁，家居服电商模特 | 三个图都一样 |
| 服装 | 米白色缎面印花家居服套装 | 颜色+面料+版型锁定 |
| 发型 | 黑色长发披肩，中分，发尾微卷 | 长度+方向+卷度锁定 |
| 色温 | 3000K 暖光 | 整片统一 |
| 光位 | 正面柔光，阴影柔和 | 避免三图光位不同 |
| 构图 | 人物全身照，居中构图 | 头顶到脚底入镜 |

## 八、常见翻车 & 修法

| 翻车现场 | 原因 | 怎么修 |
|---------|------|-------|
| 侧面图脸被虚化 | 单图三视图，模型只认一个主视觉 | 拆成独立图，加参考图 |
| 背面图出现正脸 | 提示词写了「人物」但没限定朝向 | 背面图明确写「背部朝向镜头，无面部要求」 |
| 三图肤色不同 | 没写肤色，只靠模型随机 | 加「暖白肤色，自然肤色」 |
| 服装细节不一致 | 面料描述每图不同 | 把服装描述复制成固定块，不要改动 |
| 身高比例忽长忽短 | 构图没锁死 | 统一写「全身照，头顶到脚底完整入镜」 |
| 背景风格不统一 | 只写「卧室背景」 | 加「简约卧室，暖色调，略微虚化」 |
| 参考图权重太高 | 人物姿态被主视觉肖像带偏 | 权重降到 0.8，视角提示词前置 |
| 三个图拼接成 collage | 负面提示词没加 | 加「多视图，拼贴，并排」 |

## 九、维度2 使用流程

```
1. 先确保「维度1主视觉肖像」已锁定
2. 复制固定的人物身份 + 服装 + 发型 块
3. 建 3 个 LibTV 图像节点，分别写正面 / 侧面 / 背面
4. 每个节点都上传主视觉肖像作为参考图
5. 三图的色温、光位、构图统一
6. 三图出完后逐维对照：脸 / 服装 / 发型 / 光 / 构图
7. 任何一图不通过，只改那一个角度，不要三图一起改
```
