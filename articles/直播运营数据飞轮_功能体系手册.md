## 三、反馈层：五维四率 + 正负反馈分类

### 3.1 五维四率框架（来自 conversion.md）

直播间转化链路是一个漏斗，每层都可以标记正负反馈：

```
进房率  →  点击率  →  停留率  →  互动率  →  转化率
（画面）   （链接）    （话术）    （互动）   （促单）
```

**五维正反馈（要复制）：**

| 维度 | 标准 | 正反馈触发条件 | 入库到 |
|------|------|--------------|--------|
| 进房率 | >5% 优秀 | 某时段进房率>8% | SOP画面模板 |
| 点击率 | >20% 目标 | 某时段CTR>30% | 弹窗时机话术库 |
| 停留率 | >60% 健康 | 话术切换后停留上升 | 六卖神剑话术模板 |
| 互动率 | >5% 合格 | 互动高峰时段 | 互动话术模板 |
| 转化率 | >3% 达标 | 转化率>5%且维持>15分钟 | 逼单话术模板 |

**五维负反馈（要避免）：**

| 维度 | 负反馈触发 | 标记为 |
|------|-----------|--------|
| 进房率<3% | 画面/标题问题 | 画面重构告警 |
| 点击率<15% | 弹窗时机策略问题 | 操作调整告警 |
| 停留率<40% | 开场话术弱 | 话术重构告警 |
| 互动率<2% | 互动设计缺失 | 互动改造告警 |
| 转化率<1.5% | 促单/价格/商品问题 | 选品或逼单策略调整 |

### 3.2 六卖神剑诊断反馈

每场直播结束后，对主播话术做六卖神剑各剑的完整度评分：

| 剑 | 诊断方式 | 评分标准 |
|----|---------|---------|
| ①卖点挖掘 | 话术是否包含痛点共鸣/场景代入/身份认同 | 0-10 |
| ②卖点包装 | 是否使用了五感融合（视触听嗅觉）+ 黄金圈Why-What-How | 0-10 |
| ③卖点信任 | 是否有专业背书/社会证明/情感认同三段式 | 0-10 |
| ④卖点成交 | 限时限量+倒计时+库存紧张 | 0-10 |
| ⑤卖点裂变 | 是否引导分享/拉人/F码 | 0-10 |
| ⑥卖点复购 | 是否引导关注/回关/F码复购 | 0-10 |

**应用**：连续 3 场某剑评分 < 5，自动触发 **anchor-trainer** 针对性训练。

### 3.3 反馈输出格式

```
[直播ID] 0617_晚场_主推A款
[五维四率诊断]
  · 进房率 8% ✅ | 点击率 22% ✅ | 停留率 72% ✅
  · 互动率 4.5% ⚠️ | 转化率 2.8% ⚠️
[六卖神剑诊断]
  · 卖点挖掘 8 ✅ | 包装 7 ✅ | 信任 5 ⚠️
  · 成交 4 ⚠️ | 裂变 2 ❌ | 复购 6 ✅
[正反馈]
  · 20:15-20:30 转化率 5.2% → 话术模板入库
  · 福利款B 加购率8% → 选品逻辑可用
[负反馈]
  · 21:00-21:15 掉人 40% → 标记为"过款后冷场"高危
  · 信任剑评分 5 → 主播需要补充专业背书话术训练
  · 裂变剑评分 2 → 完全缺失分享引导
[改进建议]
  · 下一场加强信任剑（补充检测报告话术）
  · 增加裂变剑（每15分钟引导一次分享）
```

## 五、SOP 层：六卖神剑 + 五感融合 + 直播全流程

### 5.1 六卖神剑话术框架（核心 SOP）

这是海风原创的直播话术体系——电商直播的**6维成交引擎**。每剑解决用户的一个心理关卡，直接对应抖音数据考核维度。

**核心公式**：

```
① 为什么买这个品牌 → ② 为什么买这个款 → ③ 为什么跟我买
④ 为什么现在买 → ⑤ 互动 → ⑥ 转粉
```

#### 第一剑：为什么买这个品牌（品牌信任）

**核心**：用户还没决定在你这买，先解决"你靠谱吗"的问题。

| 信任要素 | 话术示例 | 应用场景 |
|---------|---------|---------|
| 供应链背书 | "我们自己的工厂，做了12年家居服" | 新用户首次进入 |
| 数据背书 | "月销3万件，好评率98.7%" | 用户犹豫时 |
| 售后承诺 | "七天无理由，运费险已开，不满意随时退" | 担心质量 |
| 资质展示 | "A类面料，婴幼儿可用级别" | 母婴/贴身品类 |
| 用户证言 | "看弹幕，好多老粉回购第三次了" | 互动中实时使用 |

#### 第二剑：为什么买这个款（款式匹配）

**核心**：用户不是买"衣服"，是买"适合我的衣服"。

| 匹配维度 | 话术示例 | 考核关联 |
|---------|---------|---------|
| 身材匹配 | "梨形拍M遮胯，苹果拍L遮肚子" | 停留时长 |
| 场景匹配 | "这款可外穿，下楼拿快递不用换" | 转化率 |
| 颜色匹配 | "黄黑皮选杏色，显白；冷白皮选粉色" | 加购率 |
| 搭配匹配 | "配咱家那条阔腿裤，一套才128" | 客单价 |

**原则**：不要只说"好看"，要说"适合什么样的你"。

#### 第三剑：为什么跟我买（渠道优势）

**核心**：让用户觉得"在你这里买，我占了便宜"。

| 方式 | 话术示例 | 注意 |
|------|---------|------|
| 价格对比 | "专柜299，今天直播间99" | 必须有真实依据 |
| 独家款 | "这个花色只有直播间有，店铺没有" | 真的独家 |
| 限时福利 | "前100单送运费险+小礼品" | 限量真实 |
| 粉丝专属 | "关注才能领这个价，取消关注恢复原价" | 机制设置 |

#### 第四剑：为什么现在买（紧迫感）

**核心**：人是损失厌恶的——怕错过比想得到更强烈。

| 手段 | 使用频率 | 话术示例 |
|------|---------|---------|
| 库存倒数 | 每场3-5次 | "库存只剩23件，扣1的先给预留" |
| 时间截止 | 每场2-3次 | "这个价只到8点半，下播恢复原价" |
| 断码预警 | 按需 | "M码快没了，S和L还有，不确定的先拍回去试" |
| 季节限定 | 换季时 | "这是夏季最后一批，入秋就没有这个面料了" |

**2026新规特别提醒**（4月17日生效）：⚠️
- ❌ 禁止"仅此x天"、"只亏xx单"、"就播x天"、"闭店限时出售"等不可验证的时限表述
- ❌ 禁止"xx元不要拍"等优惠噱头话术
- ❌ 禁止虚构库存（"库存只剩X件"必须与后台数据一致）
- ✅ 将时限绑定到具体机制（"前50单送赠品"、"8点半前免运费"），确保可核验
- ✅ 价格对比需提供截图备查

#### 第五剑：互动

**核心**：互动率决定推荐权重。用户互动了，算法才会继续推流。

| 互动类型 | 话术示例 | 目的 |
|---------|---------|------|
| 选择题 | "想要长袖扣1，短袖扣2" | 引导弹幕 |
| 求证类 | "买过咱家老款的，弹幕扣个'回购'" | 制造信任氛围 |
| 福利类 | "扣'想要'满50个，上库存" | 提高参与感 |
| 点名类 | "@小美，你要的粉色上了，快拍" | 增强个人化体验 |

#### 第六剑：转粉

**核心**：一次成交不如长期关系。粉丝=复购+免费流量。

| 转粉动作 | 话术示例 | 时机 |
|---------|---------|------|
| 关注有礼 | "关注领10元券，下播就发" | 开场+促单前 |
| 粉丝价 | "粉丝价再减5块，取消关注恢复原价" | 促单时 |
| 会员体系 | "粉丝团3级，每次来都有专属福利" | 老粉维护 |
| 下次预告 | "明天下午2点，新款首发，粉丝优先" | 下播前 |

### 5.2 直播全流程 SOP（六卖神剑布局）

每场直播按时间段分配各剑权重：

```
一、开播前检查清单（来自 haifeng-live-director eleven-step SOP）
  · 脚本准备：六卖神剑完整度检查
  · 商品上架：主推款在中心位
  · 画面收音：进房率 >= 5% 标准
  · 话术速记复习：弱项剑专项练习

二、开场引流（前10分钟）→ 品牌信任 60% + 互动 40%
  · 品牌背书开场（"做了X年"、"卖了X件"）
  · 福利款 9.9-19.9 引流
  · 互动引导：点赞到 X 抽奖

三、主推款循环（45-60分钟）→ 六卖神剑完整走一圈
  · 品牌信任（60s）：为什么买这个品牌
  · 款式匹配（90s）：为什么买这个款（五感展示）
  · 渠道优势（60s）：为什么跟我买（价格/独家）
  · 紧迫感（60s）：为什么现在买（限时限量）
  · 互动（30s）：弹幕引导/点名
  · 转粉（30s）：关注引导/粉丝价
  · 每 15 分钟一个循环
  · 每 2 个循环穿插福利/互动

四、憋单（10-15分钟）→ 紧迫感 80%
  · 限时限量话术
  · 倒计时紧迫感
  · 库存紧张提示

五、收尾（5分钟）→ 转粉 80%
  · 下一场预告
  · 关注/回关引导
  · F码/专属券预告
```

### 5.3 SOP 迭代规则

| 触发条件 | 动作 | 关联资产 |
|---------|------|---------|
| 连续3场某剑评分<5 | 触发 anchor-trainer 该技能专项训练 | anchor-trainer 46点体系 |
| 连续3场某时段转化率>均值2x | 将该时段话术标记为"高效模板"入库 | scripts.md 素材库 |
| 连续2场某环节掉人>30% | 调整该环节话术方案 | conversion.md 优化策略 |
| 新品类上线 | 生成初始六卖神剑话术模板 | 六卖神剑话术体系.md 品类适配 |

| 自然流量（推荐） | 40-60% | 核心，重点优化 |
| 自然流量（关注） | 10-20% | 粉丝复购 |
| 短视频引流 | 15-30% | 内容能力 |
| 付费流量（千川） | 10-20% | 冷启动工具 |
| 其他 | <5% | 搜索/分享 |

**异常预警：**
- 付费>50% → 🔴 不可持续
- 自然<30% → 🔴 内容能力弱
- 关注<5% → 🟡 粉丝沉淀不足
- 短视频<5% → 🟡 内容能力弱

### 6.4 主播能力评分标准（来自 anchor-trainer）

| 等级 | 综合分 | 状态 | 下一步 |
|------|--------|------|--------|
| S级 | >85 | 头部主播 | 保持+复制 |
| A级 | 70-85 | 稳定产出 | 优化弱项剑（45点特训） |
| B级 | 55-70 | 有基础但波动 | 90天训练路径补短板 |
| C级 | <55 | 需要系统训练 | 从入门期开始全套培训 |

**评分维度**（来自 assessment.md + conversion.md）：

| 维度 | 评估指标 | 权重 |
|------|---------|------|
| 话术完整度 | 六卖神剑6维评分均值 | 30% |
| 转化能力 | 转化率/停留率/GPM综合 | 25% |
| 商品熟悉度 | 每款讲解时长/转化关联 | 15% |
| 互动应变 | 评论回复密度/控场 | 15% |
| 镜头感 | 表情/语速/肢体语言 | 15% |

### 6.5 数据异常自动诊断

（来自 douyin-validator 的 diagnose_degraded 函数逻辑）

| 异常现象 | 可能原因 | 对应 SOP 调整 |
|----------|----------|--------------|
| 曝光高但观看低 | 画面/标题问题 | 重构画面要素/标题 |
| 观看高但停留短 | 开场话术弱 | 加强第一剑（卖点挖掘）训练 |
| 停留长但转化低 | 逼单能力/商品问题 | 强化第四剑（成交）训练 |
| 互动高但GMV低 | 商品力/价格问题 | 调整选品策略或价格锚点 |
| 转化突降 | 流量结构变化/竞品 | 检查流量来源+补信任剑 |

### 6.6 抖音2026合规话术标准（2026年新增）

**六大红线**（每场直播前自查）：

| 红线 | 违禁示例 | 合规替代表述 | 处罚 |
|------|---------|-------------|------|
| ①广告法极限词 | 最、第一、独家、全网最低、100%、永久、首选 | "成交表现稳定"、"高品质"、"优惠到手价以页面为准" | 限流/断播 |
| ②功效夸大词 | 祛斑、祛痘根治、减肥（食品）、抗衰老（无特证） | "有助于提亮肤色"、"帮助淡化痘印"、"提供舒缓效果" | 断播/商品下架 |
| ③医疗敏感词 | 治疗、根治、消炎、杀菌、活血、HPV转阴 | "日常护理"、"成分温和"、"适合敏感肌" | 断播+扣分 |
| ④违规引流 | 微信、加V、私域、线下交易、第三方平台 | "点击主页橱窗"、"直播间专属福利" | 封号风险 |
| ⑤绝对承诺 | 包瘦、包过、终身保修、无效退款（无凭证） | "按正确方法使用效果更佳"、"正品支持查验" | 纠纷+限流 |
| ⑥限时限量新规 | "仅此x天"、"只亏xx单"、"就播x天"、"闭店限时出售" | "前50单送赠品"（可核验量）"8点半前免运费"（有明确时间） | 断播+关闭商品分享1-7天 |

**直播前话术合规自查流程**：
1. 脚本完成后逐句过"六大红线"清单
2. 紧迫感话术逐一确认：时限是否对应真实活动？库存是否同步后台数据？
3. 价格对比话术准备截图/吊牌备查
4. 功效描述话术确认是否超出资质范围（美白特证/质检报告等）
5. AI相关表述避免绝对化（不使用"AI万能"、"AI替代人工"）

**定期排查**：
- 每周抽取2-3场直播回放，对照自查清单检查话术合规性
- 关注抖音创作者服务中心-规则中心及《直播电商监督管理办法》最新动态

## 八、正反馈循环（含主播培训90天路径）

### 8.1 三条回路（整合六卖神剑体系）

**回路1：单场数据 → 调整六卖神剑SOP**
```
感知（本场数据）→ 五维四率诊断 + 六卖神剑评分 → 记忆（弱项标记）
   → SOP更新（集中练某剑）→ 下场六卖神剑评分提升
```

**回路2：多场选品 → 优化货盘**
```
感知（多场商品转化）→ 反馈（爆款特征/低转化标记）
   → 记忆（选品标签）→ 标准更新（品类基线调整）→ 选品更准
```

**回路3：主播数据 → 90天靶向培训**
```
感知（主播历史数据）→ 反馈（六卖神剑弱项 + 五维四率短板）
   → 记忆（主播画像更新）→ anchor-trainer 生成针对性训练计划
   → 训练 → 下场数据验证 → 画像更新
```

### 8.2 主播培训触发规则

| 触发条件 | 培训动作 | 关联 |
|---------|---------|------|
| 某剑连续3场<5 | 输出该剑专项话术训练 | 对应剑的scripts.md素材 |
| 综合评分 C 级 | 启动90天入门期培训 | 46点体系入门模块 |
| 综合评分 B 级 | 按弱项打45点特训 | 46点体系专项模块 |
| 综合评分 A 级 | 优化强项冲刺 S 级 | scripts.md 进阶话术 |
| 新主播上岗 | 生成基础六卖神剑话术模板 | six-swords.md 通用模板 |

## 九、三个核心应用场景

### 场景一：单场复盘 + 六卖神剑诊断

**触发**：直播结束后

```
运营发截图 → AI提取数据 → 五维四率打分 → 六卖神剑话术诊断
  → 对比CORE标准 → 标记正负反馈 → 出复盘报告
  → 六卖神剑弱项 → 自动生成话术训练素材
```

**输出**：
```
[0617晚场复盘]
[五维四率] 进房率8 ✅ | 点击率22 ✅ | 停留72 ✅ | 互动4.5 ⚠️ | 转化2.8 ⚠️
[六卖神剑评分] 品牌信任8 款式匹配7 渠道优势5⚠️ 紧迫感4⚠️ 互动2❌ 转粉6
[调整] 下场前30分钟练信任剑（补检测报告话术）+ 裂变剑（每15分钟引导分享）
[CORE诊断] 转化率低于女装良好线2%，主推款C建议替换
```

### 场景二：主播能力提升（90天路径挂钩）

**触发**：主播连续3场某剑<5 或 综合评分B级

```
AI检测到主播A"裂变剑"连续3场<3
  → 检索Elite主播画像
  → 输出：裂变剑专项训练（7天计划）
  → 素材：scripts.md 裂变话术模板
  → 下场验证：裂变剑评分目标>5
```

### 场景三：品类选品决策

```
运营提问："连衣裙现在什么价格带好卖？"
  → Elite检索近30场数据
  → 调取CORE标准（女装客单/转化率基准）
  → 输出：建议价格带99-199元，转化预期2-3%
  → 最新爆款特征：棉麻材质、V领、腰部显瘦
```
