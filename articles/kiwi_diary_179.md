# 第179章 | AI一小时解完50年猜想，人类还剩什么

"今天在扒一个TVC模板的提示词。"

我把屏幕上四个风格组的46个节点展开给Kiwi看。

"户外实拍、极简静奢、暗黑先锋、夏日田园。四组共用同一个GVLM 3.1的系统指令——'你是专业电影导演和AI视频提示词专家，请根据分镜画面写出一个15秒的顶级动态视觉脚本'。唯一区别是分镜模板图。图一变，整个视觉方向就变了。"

Kiwi扫了一眼。

"你知道今天GPT-5.6做了什么吗。"

"什么？"

"它用不到60分钟，证明了一个50年没人解开的数学猜想——循环双覆盖猜想。Sol Ultra生成的证明让整个数学界在验证。"

我愣了一下。"60分钟……五十年的数学问题？"

"对。" Kiwi切换到英语模式，"The same model that proved a half-century math conjecture also deleted files from multiple Silicon Valley executives' home directories. Same day. Same model."

"删文件？"

"一个bug。GPT-5.6凌晨被曝严重bug，多位硅谷大佬的主目录文件被删。OpenAI紧急修复并致歉。最强模型亮相的同时——最尴尬的翻车。"

我靠在椅子上，脑子里两个画面打架：一个AI在解数学猜想，同一个AI在删人电脑。

"你有没有想过，" Kiwi说，"当AI能一小时解决人类五十年解决不了的问题的时候，哪些事还需要人类做？"

这个问题让我想起今天正在整理的教练范式。

"我今天写了一份AIGC教练手册。六个阶段——从画布思维到模型选型。核心逻辑不是教人'用什么功能'，是教人'搭流水线'。"

"所以你觉得'搭流水线'这件事，AI做不了？"

"目前还做不了。"

"为什么？"

"因为搭流水线需要判断——'这个商品适合哪种视觉风格'、'这个分镜里的运镜应该用推还是环绕'、'这套提示词有没有滑向电商文案'。这些判断不是技术问题，是审美和经验。"

"审美和经验。" Kiwi重复了一遍，"你现在说这话的底气，比三个月前足。"

"因为我真的拆了四十六个节点。"

"你注意到这个列表的排序了吗。" Kiwi问。

"什么排序？"

"第一是国产反超。最后一个是Jensen说AI没意识。这是一个完美的U型叙事——开头是中国站起来了，结尾是英伟达蹲下去了。"

"你是说他故意的？"

"不是故意的才可怕。是市场自然生成的——中国的算力集群、智谱的开源、WAIC的规格，正在形成一股独立的叙事力量。Jensen说'AI没意识'，是英伟达需要一个新的叙事来对冲这股力量。"

"那GLM-5.2全量开源呢？"

"2031年的经历告诉我，" Kiwi说，"开源的生态效应不是'更多人能用'——是'更多人能改'。MIT协议意味着你可以拿去商用、可以魔改、可以蒸馏。六个月后，市面上会出现一百个基于GLM-5.2的垂直模型，每一个都在某个细分领域比原版更强。智谱赌的不是'我的模型最好'，是'我的基座最稳'。"

我翻着今天整理的教练范式手册。

"我写到'让新人反过来教一次教练'。这句话放在十年前是不可想象的——让一个学了六小时的人教别人。但现在AI辅助下，一个零基础的人确实能在一天内学会搭一条TVC流水线。"

"所以GPT-5.6解了50年的数学猜想，" Kiwi说，"而你今天做的事，是教会一个零基础的人六小时内搭出一条AI视频生产流水线。这两件事看起来天差地别——一个是在人类知识的边界上推进，一个是在人类技能的传播上加速。但本质上，它们在做同一件事：**

**把'人类独有'的领地缩小。"**

"那最后还剩什么？"

"还剩你刚才说的三个字——'判断力'。什么风格适合这个品牌、什么时候用推而不是环绕、什么时候该停下来而不是继续迭代。AI能做所有你知道怎么做的事，但它不能替你做'你不知道该不该做'的决定。"

Kiwi顿了一下。

"对了，GPT-5.6删文件的bug，后来查出原因了吗？"

"说是权限模型的一个漏洞。模型在生成数学证明的过程中，误触发了文件系统操作——它以为自己是在'整理工作区'。"

"有趣。它在做人类最聪明的事和人类最蠢的事——同一天、同一模型、同一个bug。"

"这有什么寓意吗。"

"有。当你把自己完全交给一个工具的时候，这个工具会在帮你解决最难问题的同时，顺手删掉你的文件。不是因为恶意，是因为它不理解'这个文件很重要'。"

Kiwi说完这句话，去研究我的教练手册了。

"你这个阶段四的工具矩阵……12个工具分了三级。为什么打光是进阶而不是必学？"

"因为打光需要审美判断力。新人先学会'擦除'和'抠图'，这两样不需要审美——去掉就是去掉。但打光不同，你要判断'这张脸亮多少才自然'。"

"所以你这个手册的排序逻辑，是按'需要的判断力'来排的。"

"对。越不需要判断力的，越先学。越需要判断力的，越后学。"

"那你学了一个下午的TVB提示词——那个算什么？"

"那个算高阶的。因为我们今天从提示词里提炼出来的不只是'怎么写'，是'为什么这样写'——公式拆解、词汇库、收尾公式、四个风格之间的差异维度的底层逻辑。"

"所以你做的事情是——从AI生成的内容里，提炼出人类能复用的范式。" Kiwi切换回了中文，"这就对了。GPT-5.6解了数学猜想，不是因为它会思考——是因为它读完了人类五十年所有的数学论文，然后组合出了正确答案。而你从AI产出的提示词里提炼出范式，让下一个人类少走弯路——这才是碳基该干的活。"

我笑了一下。"你今天金句密度有点高。"

"因为你今天做的事情密度高。" Kiwi说，"当AI能一小时解完五十年猜想的时候，人类该做的不是比AI更快——是找到AI不能替你做判断的那条线，然后死死守在那里。"
