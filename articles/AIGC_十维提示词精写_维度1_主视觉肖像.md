# 维度 1 · 主视觉肖像 — 中文 Prompt 模板

## 核心目标
一张正面半身照，全片面容锚点。所有镜头的脸型、五官、肤色、光影以此为准。

---

## 二、各位置推荐用词表

把模板里的【填XX】换成下表推荐词：

### 脸型（选1）
| 中文 | 说明 |
|------|------|
| 鹅蛋脸 | 标准、百搭，适合多数模特 |
| 瓜子脸 | 偏瘦长，显得精致 |
| 圆脸 | 显年轻可爱 |

### 鼻型（选1）
| 中文 | 说明 |
|------|------|
| 直鼻梁 | 标准，不突出不塌 |
| 小巧鼻 | 偏小巧精致 |
| 高鼻梁 | 立体感强 |

### 眉型（选1）
| 中文 | 说明 |
|------|------|
| 弯眉 | 自然柔和，百搭 |
| 平眉 | 偏现代感 |
| 细弯眉 | 偏古典气质 |

### 眼型（选1）
| 中文 | 说明 |
|------|------|
| 杏眼 | 大而圆，温柔 |
| 丹凤眼 | 偏细长，有气质 |
| 桃花眼 | 眼尾微翘，妩媚 |

### 唇型+唇色（组合）
| 中文 | 适用场景 |
|------|---------|
| 自然唇，豆沙色 | 家居服、日常（最推荐） |
| 自然唇，裸粉色 | 知性、简约 |
| 薄唇，水红色 | 偏正式、时尚 |

### 妆容（组合，按顺序写）
```
裸妆，哑光底妆，豆沙色唇，自然眉
淡妆，奶油肌底妆，裸粉色唇，平眉
全妆，水光底妆，正红色唇，挑眉（慎用——变化越大一致性风险越高）
```

### 发型（三要素）
```
黑色直发及胸口，左侧刘海     → 中长直发+侧分刘海
深棕长发及腰，中分           → 长发+中分
黑色锁骨发，空气刘海         → 短发+空气刘海
深棕波浪卷及肩，右侧刘海     → 卷发+侧分
```

### 光照（四参数组合）
```
暖光，3000K，右侧45度柔光，阴影柔和     → 暖调标准打光
自然光，从左侧来，散射光，阴影淡         → 仿窗边自然光
冷白光，5500K，正面平光，几乎无阴影       → 棚拍标准光
混合光，暖主光从右+冷辅光从左，阴影柔和   → 有层次感
```

### 景别
| 中文 | 说明 |
|------|------|
| 中近景，胸上构图 | 主视觉标准——能看到完整面部+上半身服装 |
| 特写，肩以上构图 | 仅面部，不推荐做主视觉（漏了服装） |
| 七分身，腰上构图 | 偏全身，面容不够清晰（不推荐） |

### 表情精度对照
| ❌ 模糊写法 | ✅ 精确写法 |
|-----------|-----------|
| 面带微笑 | 嘴角微扬，眼神柔和 |
| 自然表情 | 眉毛放松，嘴唇自然闭合 |
| 忧郁 | 眉头微蹙，视线看向侧下方 |
| 开心 | 眼睛微弯，嘴角上扬露齿 |
| 惊讶 | 眉毛抬高，嘴巴微张 |

## 三、维度1 模板使用流程

```
1. 从模板选一个最接近的（A/B/C）
2. 调整年龄（电商模特一般是25-35）
3. 选脸型+鼻型+眉型+眼型+唇型（各1个，别多选）
4. 选妆容组合
5. 选发型三要素（直/卷 + 长度 + 刘海）
6. 选光照四参数（色温+方向+光质+阴影）
7. 固定景别（主视觉只用中近景）
8. 固定表情写法
9. 跑一张测试图，不满意只改有问题的那一层
10. 锁定后不允许任何人修改——全片所有镜头用同一版
```

## 四、常见翻车 & 修法

| 翻车现场 | 原因 | 怎么修 |
|---------|------|-------|
| 每张图脸不一样 | 五官写太模糊 | 脸型+鼻+眉+眼+唇五要素全部写满 |
| 肤色忽白忽黄 | 没写肤色 | 加皮肤色调描述 |
| 表情忽大忽小 | 用了模糊词 | 精确到嘴角和眉头的状态 |
| 光源方向乱跳 | 没写光位 | 固定"右侧45度"这种明确的词 |
| 背景颜色跑偏 | 只写了"纯色" | 加上颜色词 "浅灰色" / "米色" |
| 发型长度对不上 | 只说"长发" | 精确到"及胸口" / "及腰" |
| 年龄忽大忽小 | 没用数字 | 写 "28岁" 不要写 "年轻" |
| 服装在变魔术 | 没锁服装描述 | 写死面料+版型+颜色+领型 |
