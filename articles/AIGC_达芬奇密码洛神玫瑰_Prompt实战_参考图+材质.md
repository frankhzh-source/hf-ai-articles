## 二、LibTV 参考图节点结构

```

  → 商品整体外观参考（印花/版型/配色）
```

## 七、十维一致性应用检查

| 维度 | 本商品处理要点 |
|------|---------------|
| 1 主视觉肖像 | 用模特图作为参考图锁定人物 + 服装 |
| 2 三视图 | 拆3个节点：正面/左侧面/背面，每图传参考图 |
| 3 表情设定 | 固定「柔和微笑」，不要每图换表情 |
| 4 面部细节 | 自然妆、豆沙唇、中分长发，保持一致 |
| 5 服饰材质 | 重点写「聚酯纤维混纺、微弹、垂坠、丝滑哑光」 |
| 6 手部与姿态 | 自然垂放或轻贴裤缝，避免复杂手势 |
| 7 道具特写 | 无道具，忽略 |
| 8 色彩参考 | 浅色底 + 粉色/玫红印花 + 3000K暖光 |
| 9 剪影/比例 | 全身照，居中，头顶到脚底完整入镜 |
| 10 角色信息 | 28岁亚洲女性，电商家居服模特，慵懒居家风 |

## 八、流程建议

1. 先上传 3 张参考图到 LibTV 图像节点
2. 从「维度1 主视觉肖像」开始跑，验证人物+服装一致性
3. 主视觉锁定后，复制到「维度2 三视图」3 个节点
4. 跑材质特写，确认印花和面料质感
5. 三视图+材质都通过，再进入视频分镜生成
